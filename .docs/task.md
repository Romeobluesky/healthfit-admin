###

2. Nginx + mysql + pm2

3. 다크모드 라이트모드 시스템모드
4. 권한 지정 : 관리자 -> 10, 파트너A -> 9, 파트너B -> 8
5. 페이지 구성
  1. 로그인 / 로그아웃
  2. 메인 대시보드
  3. 고객관리
    - 간편인증고객리스트
      - 리스트 타이틀
        - 번호, 이름, 전화번호, 생년월일, 성별, 등록일, 리포트보기, 삭제(관리자만)
        - 리포트보기
          - 클릭시 내용 모달 (추후제작)
          - 모달 내용 pdf 파일 저장 기능 (추후제작)
    - 일반고객리스트
      - 리스트 타이틀
        - 번호, 이름, 전화번호, 생년월일, 성별, 등록일, 설문내역보기, 삭제(관리자만)
        - 리포트보기
          - 클릭시 내용 모달 (추후제작)
          - 모달 내용 pdf 파일 저장 기능 (추후제작)

  4. 파트너관리
    - 리스트
      - 번호, 파트너명, 아이디, 패스워드, 전화번호, 생성일, 상태(미승인, 승인, 중지)
    - 등록, 수정, 삭제 기능

  5. 서비스코드관리
    - 생성
      - 형식(숫자) :  4자리 - 4자리 - 4자리 (예: 1234-1234-1234)
      - 파트너별 지정 생성 (관리자포함)
    - 리스트
      - 코드
      - 권한 (관리자, 파트너A, 파트너B)
      - 파트너명
      - 파트너 아이디
      - 상태 (미활성 Inactive, 활성 Activation, 사용중 Use, 미사용 Notused)
